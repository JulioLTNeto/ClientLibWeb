    ◆ tInformation

    typedef struct payload* tInformation

    Struct de Payload.

    O typedef tInformation é uma struct de payload (carga) e deve ser utilizada toda vez que uma nova informação precisar ser recebida ou enviada (para entender melhor, verifique a aplicação teste). A informação enviada sempre precisará, obrigatoriamente, preencher essa struct através das funções de Set e Add Information.

    A informação sempre será recebida em forma dessa struct, e o programador deve destrinchá-la através das funções de Get Information e Get Property.

    ◆ ClientLibShutdown()

    void ClientLibShutdown ( )

    Desliga a ClientLib.

    É recomendado que, no fim do programa, ou quando não quiser mais usar a ClientLib no programa, o programador use a função ClientLibShutdown. Esta função garante que todas as pré-configurações da biblioteca sejam desativadas, e memória ocupada seja liberada.
